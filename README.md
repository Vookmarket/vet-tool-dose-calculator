# 薬用量クイック計算

体重と薬剤名（または症状）を入力すると、犬猫の一般的な薬用量をAIが計算して日本語で表示します。

**Vet Tools Daily** シリーズ #1 | 対象: 獣医師・看護師 | 難易度: ★☆☆

## できること

- 犬猫の体重に基づいた薬用量の自動計算
- 薬剤名からの用量検索（mg/kg → 実投与量）
- 症状からの一般的な処方薬の提案

## 必要なもの

- Python 3.10以上
- Ollama + Qwen3.5モデル（ローカルで動作、インターネット不要）

## インストール

```bash
git clone https://github.com/Vookmarket/vet-tool-dose-calculator.git
cd vet-tool-dose-calculator
pip install -r requirements.txt
```

### Ollamaのセットアップ

```bash
# Ollamaをインストール（https://ollama.com）
# Qwen3モデルをダウンロード
ollama pull qwen3.5:9b-q8_0
```

## 使い方

```bash
python3 tool.py
```

### 実行例

```
$ python3 tool.py

⚠️ 注意: このツールの出力はAIによる参考情報です。
臨床判断は必ず獣医師が行ってください。

==================================================
  薬用量クイック計算 (Dose Calculator)
  Vet Tools Daily #1

終了するには Ctrl+C または 'q' を入力してください。

動物種を入力してください（犬 / 猫）: 犬
体重を入力してください（kg）: 10
薬剤名または症状を入力してください: アモキシシリン

計算中...

■ アモキシシリン（AMPC）
  用量: 10〜20 mg/kg
  投与経路: PO（経口）
  頻度: BID（1日2回）
  ──────────
  💊 計算結果: 100〜200 mg（1日2回）

【注意事項】
・消化器症状（嘔吐、下痢）に注意
```
